关于本项目

本项目基于卷积神经网络（CNN）构建手势数字识别系统，使用 PyTorch 深度学习框架训练多层卷积模型，通过 Flask 提供 Web API 服务。集成数据采集、模型训练、模型更新、实时识别、数据分析等完整的 AI 工程流程。
