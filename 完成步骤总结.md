## 二、模型 — CustomNet v2

**文件**: [model.py](nndl_project-master/model.py)

### 2.1 架构

| Block | 结构 | 输出尺寸 |
|:------|:-----|:--------|
| 输入 | RGB 图片 | 3×64×64 |
| Block 1 | 2×Conv(32,3×3)+BN+ReLU+MaxPool(2) | 32×32×32 |
| Block 2 | 2×Conv(64,3×3)+BN+ReLU+MaxPool(2) | 64×16×16 |
| Block 3 | 2×Conv(128,3×3)+BN+ReLU+MaxPool(2) | 128×8×8 |
| Block 4 | 2×Conv(256,3×3)+BN+ReLU+MaxPool(2) | 256×4×4 |
| 池化 | AdaptiveAvgPool1d(512) | 512 |
| 分类头 | Dropout(0.5)→Linear(512→256)→ReLU→Dropout(0.3)→Linear(256→10) | 10 |

**参数量**: 4,821,194（约 4.8M）  
**特点**: 每阶段双卷积 + BatchNorm + AdaptiveAvgPool + 双层 Dropout

### 2.2 数据增强

```python
RandomRotation(25°) → RandomAffine(translate=0.12, scale=0.8~1.2, shear=10°) 
→ RandomPerspective(0.15) → ColorJitter(0.25) → GaussianBlur(3) 
→ RandomGrayscale(0.1) → Resize(64,64) → ToTensor → Normalize(0.5, 0.5) 
→ RandomErasing(0.15)
```

### 2.3 训练配置

| 参数 | 值 |
|:-----|:--|
| Batch Size | 128 |
| Epoch | 20（10 基础 + 10 微调） |
| 学习率 | 5e-4 |
| 优化器 | AdamW (weight_decay=1e-4) |
| 调度器 | ReduceLROnPlateau (patience=3, factor=0.5) |
| 损失函数 | CrossEntropyLoss |
| 设备 | **CUDA GPU** (RTX 4060) |

```
┌─────────────┬────────────────────────┬─────────────┐
│  340px      │        1fr             │  340px      │
│             │                        │             │
│  上传图片    │      图片预览           │  识别结果    │
│  (180px框)  │   (铺满预览区)          │  圆环 → 置信度│
│             │   object-fit:contain   │  → 四格信息  │
│  选择照片    │                        │  → 概率条    │
│  (52px按钮) │                        │             │
│             │                        │  ─────────  │
│  示例图片    │                        │  重新识别    │
│  (0-5按钮)  │                        │  重新上传    │
│             │                        │             │
└─────────────┴────────────────────────┴─────────────┘
       540px 固定高度，顶部对齐，底部对齐
```

**关键 CSS**:
- Grid: `340px 1fr 340px`，`align-items:start`
- 三个格子：`height:540px` 固定，互不撑大
- 预览框：`flex:1`，图片 `position:absolute; width:100%; height:100%; object-fit:contain` 完全铺满
- 识别结果：`flex + space-evenly` 垂直均匀分布，按钮固定在底部（分隔线隔开）

## 六、混合训练系统

**文件**: [train_mixed.py](nndl_project-master/train_mixed.py)

### 6.1 数据处理

- 原始数据（4,402 张，统一背景）+ 实拍照片（82 张，各种背景）
- 实拍照片重复 3 倍以平衡类别权重
- 实拍照片使用更强数据增强（旋转 35°、透视 0.2、色彩抖动 0.3）
- 验证集：从训练集随机取 10%

### 6.2 训练流程

1. 基础训练（10 Epoch）：原始数据 + 实拍照片混合
2. 实拍微调（10 Epoch）：继续在混合数据上训练
3. 每 2 轮验证一次，最佳模型自动保存至 `models/model.pkl`
4. 训练完成写入 `__TRAINING_COMPLETE__` 标记

### 6.3 进度写入

- 终端输出同步写入 `train_progress.txt`
- 前端通过 `/api/train/progress` 读取并解析

## 八、问题修复记录

| 问题 | 原因 | 修复 |
|:-----|:-----|:-----|
| `ModuleNotFoundError: torch` | Python 3.11 未装 PyTorch | pip install torch torchvision |
| `mat1 and mat2 shapes mismatch` | 上传图片尺寸 ≠ 64×64 | 添加 `Resize((64,64))` 到预处理 |
| 端口 5000 占用 | 旧进程 / Flask debug 子进程 | `taskkill //F //IM python.exe` + port fallback |
| UnicodeEncodeError GBK | Windows 终端不支持 emoji | 替换为 ASCII 字符 |
| 实拍照片文件名空格 | CustomDataset 空格分割解析失败 | 重命名去除空格和括号 |
| torchvision 版本不兼容 | torch 2.6.0+cu124 vs torchvision 0.27.1 | `--force-reinstall --no-deps torchvision==0.21.0+cu124` |
| 训练无 epoch 输出 | 训练进程卡住 | kill 后重启 |
| 预览图片不显示 | CSS `display:none` 覆盖 | 移除 display:none，用 absolute 定位处理占位文字 |
| 预览框被图片撑大 | `flex:1` + `height:100%` 导致无限扩展 | 固定 `height:540px` 在卡片级别，内部用 `flex:1` 和 `position:absolute` |
| 图片不铺满预览框 | `max-width/max-height` 不强制拉伸 + `align-items:center` 阻止 | 改为 `position:absolute; width:100%; height:100%; object-fit:contain` |
| 三列高度不一致 | `align-items:start` 导致各自自然高度 | 统一 `height:540px` 固定高度 + `align-items:start` 顶部对齐 |
